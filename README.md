# ⚽ Football Hub - Dream Squad 🏃‍♂️

¡Bienvenido a **Football Hub**, el portal definitivo de tácticas y gestión de plantillas de fútbol! Este proyecto es una aplicación web interactiva desarrollada en **React** y construida con **Vite**, diseñada para explorar perfiles, estadísticas analíticas y configuraciones estratégicas de los mejores futbolistas del mundo.

---

## 🚀 Características Principales

*   **Catálogo de Futbolistas Dinámico**: Visualización de tarjetas de jugadores con estadísticas clave (Media, Posición, Equipo) renderizadas de forma reactiva.
*   **Simulación de API Asíncrona**: Sistema de carga optimizado que implementa retrasos de red simulados (`setTimeout` y promesas) para manejar estados de carga (`loading`) y manejo de errores estables.
*   **Reproductor de Audio Integrado**: Interfaz con música de fondo interactiva para mejorar la experiencia de usuario mientras se navega por la pizarra táctica.
*   **Diseño Moderno y Responsivo**: Interfaz optimizada para dispositivos móviles y de escritorio utilizando componentes estilizados con una paleta oscura y detalles deportivos en verde esmeralda.

---

## 🛠️ Tecnologías Utilizadas

*   **Frontend**: React.js (Hooks: `useState`, `useEffect`)
*   **Enrutamiento**: React Router
*   **Estilos**: CSS3 moderno / Framework de Estilos Responsivos
*   **Entorno de Desarrollo**: Vite
*   **Control de Versiones**: Git & GitHub
